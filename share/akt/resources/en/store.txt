store: store files, directories or read standard input in the keystore

Usage: akt store <keystore>.akt [-r] <files> <directories>
       akt store <keystore>.akt -- name

  This command can store files or directories recursively in the
  keystore.

  The store command can also be used as a target for a pipe
  command.  It reads the standard input and stores the content
  which is read in the keystore.
